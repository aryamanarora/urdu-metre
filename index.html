Urdu metre

This is a small tool for automatically doing metrical scansion of Urdu poetry, based on the guide by Pritchett and Khaliq (2003). All possible variant readings are not fully supported yet but this will make understanding metre a little easier by automatically checking complex variants like word-boundary grafting. Paste a text from Rekhta below (or use an IAST-esque transliteration) to generate scansion.

Made by Aryaman Arora somewhat uncomfortably using vanilla JavaScript.
